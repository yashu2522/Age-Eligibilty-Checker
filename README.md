# Age Eligibility Checker 🎂

A simple web application that checks a user's eligibility for various activities based on their age.

This project was built as part of revising JavaScript fundamentals, especially conditional logic and DOM manipulation.

## 🚀 Tech Stack

* HTML5
* CSS3
* JavaScript (DOM Manipulation)

## ✨ Features

* Accepts user age input
* Validates age (0–120 range)
* Checks eligibility for:

  * Voting
  * Driving
  * Alcohol consumption
* Displays multiple results dynamically
* Simple and user-friendly interface

## 📚 What I Practiced

* Handling user input and validation
* Using conditional statements (`if-else`)
* DOM manipulation with `getElementById`
* Dynamically updating content using `innerHTML`
* Working with multiple conditions in a single flow

## 📌 Future Improvements

* Improve UI with better styling
* Add country-specific rules (different age limits)
* Show results with icons or colors
* Add real-time validation

## 📁 Project Status

Completed as a revision project to strengthen JavaScript logic building and real-world condition handling.
